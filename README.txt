# P6-Piiquante

## Instruction 

1. creer un dossier images dans le dossier backend
2. creer un fichier .env dans le dossier backend puis copy/paste les variables d'environnement du fichier env.txt dans le fichier .env
3. npm install dans le dossier backend
4. git clone du frontend => https://github.com/OpenClassrooms-Student-Center/Web-Developer-P6 dans le dossier géneral 
4. renommer ce nouveau dossier "frontend" puis npm install dans celui-ci
5. npm start dans le dossier frontend et backend

**Enjoy**

Note :  Si lors du npm start dans le dossier backend l'erreur querySrv ECONNREFUSED _mongodb.tcp.oc-p6.phsikdj.mongodb.net at QueryReqWrap.onresolve apparait, commentez le premier bloc mongoose.connect et décommentez le deuxieme bloc. Faites npm start à nouveau.
